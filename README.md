🎯 Estudos de Dart - Aula 01
Anotações rápidas sobre a transição de Python para Dart e os primeiros desafios no terminal.

📝 O que rolou:
Tipagem Estática: Diferença entre int, double e bool (nada de 0 e 1 para booleano aqui).

Entrada de Dados: Aprendi a usar stdin.readLineSync()! e como converter String para número.

Saída Formatada: Uso de \n para pular linha e \$ para imprimir o cifrão de dinheiro sem bugar a variável.

Funções com Tipo: No Dart, as funções têm tipo (void, double, int) e não são apenas def.

Lógica de Arredondamento: Uso do .truncate() para criar um sistema de cupons (gasto / 50).

🛠️ Desafios Superados:

Git Sync: Resolvido o erro de push rejected com pull e entendi como organizar a branch de exercícios.
