🧮 Projeto: Calculadora Dart (Aula Guiada Da ALURA)
Este repositório contém o código desenvolvido durante a aula prática, seguindo o passo a passo do professor para entender a estrutura de um app CLI (Command Line Interface).

🚀 O que foi construído:
Entrada de Dados: Captura de números e da operação matemática via stdin.

Estrutura de Controle: Implementação de if / else (ou switch) para selecionar a operação.

Funções Matemáticas: Separação da lógica de soma, subtração, multiplicação e divisão.

Tratamento de Erros: Como evitar que o programa quebre ao digitar letras ou dividir por zero.

🧠 Aprendizados da Aula:
Organização de Código: Como o professor estrutura as funções fora do main.

Tipagem de Retorno: Prática de funções que retornam double para cálculos precisos.

Fluxo de Execução: Entendimento de como o Dart lê o código de cima para baixo.

🛠️ Tecnologias:
Dart SDK

Biblioteca dart:io
