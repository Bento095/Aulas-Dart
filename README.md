# Meu App

## Trajetória do Repositório
Este repositório documenta o processo de aprendizado em Dart de um estudante que vinha de Python e começou a explorar o ecossistema Dart/Flutter.

A trajetória inclui:
- aulas e exercícios práticos feitos em arquivos separados dentro da pasta `bin/`;
- experimentos com entradas e saídas no terminal usando `stdin.readLineSync()`;
- prática de tipagem estática em Dart, com `int`, `double`, `bool` e funções tipadas;
- estudos sobre estruturas de controle, operadores e lógica de programação;
- uso de testes básicos em `test/meu_app_test.dart` para validar comportamentos.

## Finalidade do Repositório
O objetivo principal é servir como um caderno de estudos e projeto de apoio para:
- fixar conceitos básicos de Dart;
- entender a diferença entre programação em Python e Dart;
- criar pequenos programas funcionais no terminal;
- organizar exercícios por aula e comparar o aprendizado ao longo do tempo.

## Estrutura do Projeto
- `bin/` — códigos de exemplo e exercícios em Dart, como atividades de aula e pequenos programas.
- `lib/` — arquivos de suporte para o aplicativo principal.
- `test/` — testes unitários que garantem que o código funciona corretamente.
- `pubspec.yaml` — dependências e configurações do projeto Dart.

## Como usar
1. Instale o SDK Dart.
2. Abra o terminal na pasta do projeto.
3. Execute um arquivo Dart com:
   ```bash
   dart run bin/meu_app.dart
   ```
4. Rode os testes com:
   ```bash
   dart test
   ```

## Aprendizados principais
- transição de Python para Dart;
- uso de tipagem estática, funções e classes;
- leitura e conversão de dados do usuário;
- organização de código em pastas `bin`, `lib` e `test`;
- controle de versão com Git e resolução de conflitos de push/pull.

## Observações
Este repositório é uma base de estudos e pode ser ampliado com novos exercícios e aplicações à medida que o aprendizado avança.
